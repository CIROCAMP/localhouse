## 🎯 Key Design Principles

1. **Elegance through Simplicity**: Minimal elements, maximum impact
2. **Generous Whitespace**: Breathing room between sections
3. **Consistent Typography**: Hierarchy through size and weight, not color
4. **Gold Accents**: Used sparingly for emphasis (dividers, buttons, highlights)
5. **Smooth Animations**: Subtle hover effects, fade-ins on scroll
6. **Mobile-First**: Responsive design that scales beautifully
7. **Authentic Imagery**: Real photos of The Local House (no AI-generated images)
8. **Storytelling**: Every section builds the narrative of Italian authenticity + Miami vibrancy

## 🎨 Color Usage Guide

| Element | Color | Usage |
|---------|-------|-------|
| Headers (h1, h2, h3) | Navy (#1A3A52) | Main headlines, section titles |
| Body Text | Charcoal (#2C2C2C) | Paragraphs, descriptions |
| Background | Cream (#F5F1E8) | Main background, light sections |
| Accents | Gold (#D4AF37) | Divider lines, buttons, highlights |
| Buttons (Primary) | Gold bg, Navy text | Main CTAs |
| Buttons (Secondary) | Navy bg, Cream text | Alternative CTAs |
| Buttons (Outline) | Navy border, Navy text | Tertiary CTAs |
| Hover States | Darker Navy or lighter Gold | Interactive feedback |

---

## 📐 Spacing System

- **Section padding**: py-16 (small), py-20 (medium), py-24 (large), py-32 (hero)
- **Container**: max-w-6xl for full-width sections, max-w-4xl for content sections
- **Gap between elements**: gap-4 (tight), gap-6 (normal), gap-8 (spacious), gap-12 (generous)
- **Margin bottom**: mb-4 (tight), mb-6 (normal), mb-8 (spacious)

## ✨ Elegant Touches

- Smooth scroll behavior
- Subtle shadows (not harsh)
- Rounded corners: 2px (minimal, elegant)
- Letter-spacing on headers: 0.5px
- Line-height: 1.7 for body text
- Font weights: 600-700 for headers, 400-500 for body
- Hover animations: 300ms ease-in-out
- Gold divider lines: 4rem width, 0.25rem height, rounded ends
